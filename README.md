# Bienvenidos a CodiPrompt 

## ¿Qué hago para ejecutar ?
primero se entra como super usuario
````bash
sudo su
````

Antes de empezar no olvides instalar estas dependencias en sudo su: 
````bash
    pip install --upgrade pip
    pip install web.py
    pip install dotenv
    pip install requests
    pip install plotly
    pip install bcrypt
````

despues es:
````bash
python3 main.py 80
````
